--- layout: page title: Overview of MOE's basics group: navigation --- In this section, the basic controls in MOE are explained. Subsequently, you will load a provided receptor structure, which was modeled in a virtual screening approach. Since this receptor comes without a ligand, you will learn how to identify the binding site in order to guide the docking process.

Step 1 - Opening the receptor file

  • [MOE] → File → Open → aa1r_v1.pdb → OK

    • Step 2 - Basic MOE controls

    During this example you'll work with three main windows of MOE, namely the main window itself, termed [MOE], the database viewer [DBV] and the sequence editor [SEQ]. Two important components of the MOE window are the "Right Hand Side" button bar [RHS] and the footer [Footer], where you find a lot of helpful shortcuts.

  • [MOE] → System
    • In this windows, all objects loaded into MOE are shown
  • [MOE] → Help → Mouse
    • This will print the current mouse settings.
  • [MOE] → Help → Contents...
    • More examples are available within Help.

    Now try to become familiar with the MOE environment. For example:
  • [MOE] [Footer] → Atoms/Ribbon
    • Those options allows you to change the appearence of atoms.
  • [MOE] [RHS] → Hide/Show
    • This option allows to show or hide atoms/residues of the system.
  • Rotate/translate the view, set center of rotation

    • Step 3 - Modelling caffeine to the binding site

    We will load a crystal structure of the Adenosine A2A receptor bound to caffeine. Since these two structures have a similar binding site, we expect caffeine to bind in a similar manner to both of them. To get a (very rough) idea of this binding mode, we're going to superpose the two proteins and copy the position of caffeine from the A2A receptor structure to our model. Now, open the sequence editor.
  • [MOE] → [SEQ]
    • Load the adenosine A2A structure. The sequence will appear in the [SEQ] window and the structure in the [MOE] window. Loading this structure via [MOE] would have worked, too.
  • [SEQ] → File → Open → 3RFM.pdb → OK
    • Now superpose the two structures. This structural superposition requires MOE to calculate a sequence alignment first.
  • [SEQ] Alignement → Superpose...

  • ... → Superpose → Yes → Close

    • Investigate the aligned structures and their binding sites carefully. We can easily see, that caffeine interacts with the asparagin as mentioned.
    Now, delete the protein part from the crystal structure, maintaining the small molecule.
  • [SEQ] → right-click 3RFM.pdb protein chain → Delete → OK
    • MOE still considers the ligand and the receptor to be independent objects. You can check that in the system:
  • [MOE] → System

    • Rename the tag and the chain to make MOE considering caffeine to be part of the model.
  • [SEQ] → right-click CFF chain → Rename → "caffeine"
  • [SEQ] → right-click caffeine tag → Rename → "aa1r_v1.pdb"

    • n.b. Caffeine has now become part of our model, you can check again the system.

    Now, MOE allows us to use some convenient shortcuts, that were not possible before.
  • [MOE] [RHS] → Show → Pocket
  • [MOE] [RHS] → Select → Ligand
  • [MOE] [footer] → Atoms → select your favorite depiction

    • n.b. Changing the way things look is possible via the system tool too.
